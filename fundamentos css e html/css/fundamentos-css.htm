Box Model

Como funciona o display:
Block:
- ocupa todo o bloco
- width e heigh são aplicados
- padding, margin e border funcionam por completo elementos com display block padrão:
div, main, header, section, p, h...

Inline:
- ocupa apenas o espaço do elemento
- fica em linha
- width e heigh não são aplicados
- padding, margin e border aplicam-se apenas nas horizontais
*usa-se inline-block para resolver*
- gitelementos com display block padrão:
div, main, header, section, p, h...

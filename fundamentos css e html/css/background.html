🖼️ Background Size & Repeat

Todos os exemplos usam a mesma imagem. Veja como cada propriedade muda o resultado.

📐 background-size: cover vs contain

Controla como a imagem se encaixa dentro do elemento.

🔴 cover

background-size: cover
A imagem preenche tudo, sem deixar espaço vazio.
Se sobrar, as bordas são cortadas.

👉 Ideal para banners e fundos de página.

🟢 contain

background-size: contain
A imagem aparece inteira, sem cortar nada.
Se a proporção for diferente, sobram espaços vazios nas bordas.

👉 Ideal para logos e ícones que não podem ser cortados.

🔁 background-repeat

Controla se (e como) a imagem se repete para preencher o espaço.

repeat

background-repeat: repeat
Padrão do CSS. A imagem se repete em todas as direções.

👉 Útil para texturas e padrões decorativos.

no-repeat

background-repeat: no-repeat
A imagem aparece uma única vez.

👉 O mais usado no dia a dia, especialmente junto com cover.

repeat-x

background-repeat: repeat-x
A imagem se repete só na horizontal.

👉 Útil para bordas decorativas no topo ou rodapé.

repeat-y

background-repeat: repeat-y
A imagem se repete só na vertical.

👉 Útil para bordas laterais decorativas.
💡 Combinação mais usada no dia a dia:
background-size: cover + background-repeat: no-repeat + background-position: center

Essa combinação garante que a imagem preencha o elemento inteiro, não se repita e fique centralizada. É o padrão usado em praticamente todo banner, hero e fundo de seção. ✅

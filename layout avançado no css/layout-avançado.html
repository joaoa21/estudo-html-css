Position

o padrão é static

Relative

- vai manter o normal flow
- deslocamento relativo a ele mesmo
- z-index: controla a "profundidade" do elemento - posição: bottom, top, right, left

Absolute

- Ele sai completamente do fluxo — os outros elementos agem como se ele não existisse.
- Se posiciona em relação ao pai mais próximo com position: relative. Se não tiver nenhum pai relative, vai usar o body inteiro como referência.

Fixed

fica fixo na tela

Sticky

fixo e relative - ele fixa na tela depois da rolagem
